# Agent Orientation

Start with the current North Star map before doing repo work.

## Read First

1. [Vision - North Star.md](Vision%20-%20North%20Star.md)
   - The aggressive vision: find the geometry whose shadows are physics, then
     prove it or break it.
2. [Method - Research Program Guidelines.md](Method%20-%20Research%20Program%20Guidelines.md)
   - The methodology: type claims, declare projections and capabilities, run
     absorber passes, demote weak residue, and keep the vision falsifiable.
3. [Lead Research Line - Time as Finality.md](Lead%20Research%20Line%20-%20Time%20as%20Finality.md)
   - The lead research line: test record stabilization, finality, and temporal
     direction inside the broader geometry vision.

## Working Rule

The vision is allowed to be aggressive. The research program is not.

When adding or reviewing work, keep this split clear:

```text
Vision - what might be true.
Method - how to prevent self-deception.
Research lines - concrete ways to prove, weaken, absorb, or kill the vision.
```

The old root `NORTH-STAR.md` is archived under `Candidate North Star/archive/`
and should be treated as historical context plus the source for the current
lead research line.
