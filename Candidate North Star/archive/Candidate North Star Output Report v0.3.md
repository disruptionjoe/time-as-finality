## What Changed From v0.2

v0.2 made the candidate safer.

v0.3 makes it more honest and more ambitious at the same time.

The biggest novelty downgrade is global:

```text
the bare factorization failure is old
```

The draft now names the strongest absorbers:

- Blackwell / Le Cam / decision sufficiency;
- concurrency and process semantics;
- abstract interpretation and complete abstractions;
- POMDP belief-state sufficiency;
- resource convertibility;
- causal and control theory;
- database view/query determinacy;
- quantum information.

The draft therefore frames Capability Projection as:

```text
a cross-domain audit surface
```

not as a new theorem.

## Major Additions

## 1. Two Forms Of Absorption

v0.3 distinguishes:

```text
state-enrichment absorption
native-theory absorption
```

Formal residue must survive both.

## 2. Canonicity And Gerrymandering

v0.3 adds:

```text
gerrymandered_capability
```

for cases where `Cap` is predeclared but not domain-natural.

It also adds:

```text
canonical_residue
```

as the strongest target.

## 3. Database Integration

Database theory is now treated as the cleanest finite executable absorber,
especially through view/query determinacy, provenance, MVCC, access control,
materializations, indexes, distributed metadata, and vector retrieval.

Database examples are framed as finite witnesses and preservation controls,
not novelty evidence.

## 4. Physics Strongman

The physics sections are no longer just safety labels.

Each one now instantiates:

```text
Y / X / pi / Cap / K / equivalence / support / kill condition / absorber /
not claimed
```

This keeps the physics bold without making it evidential support.

## Bottom Line

v0.3 preserves the Candidate North Star, but with sharper humility:

```text
The slogan is old.
The audit discipline may still be useful.
Canonical residue is the hard target.
```
