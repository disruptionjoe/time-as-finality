# Emergence Laboratory Specification

## Research Question

Which parts of record finality arise from local dynamics, which depend on the
substrate, and which are only analogies?

The laboratory replaces the hand-authored record tokens of T1 with records
generated by explicit local-update rules.

## Substrates

The first comparison uses two finite, periodic, binary cellular systems:

1. an elementary cellular automaton (ECA), with Rule 110 as the primary
   irreversible example;
2. a second-order cellular automaton built from the same local rule:

   ```text
   previous, current -> current, rule(current) XOR previous
   ```

   This map is reversible because:

   ```text
   previous = rule(current) XOR next
   ```

Reversibility is also checked exhaustively on the finite state space. It is
not accepted from the construction alone.

## Intrinsic Record Definition

A later cell contains a record of a seed cell when a counterfactual
intervention on that seed changes the later cell while the update rule,
boundary condition, and all other initial bits remain fixed.

For seed event `s` and later event `e`:

```text
record(s, e) = state(e | flip(s)) XOR state(e | baseline)
```

This definition requires no external object recognition. It does not label
gliders, particles, memories, or meaningful patterns.

## Causal Graph

Events are cell updates. A directed edge from parent event `p` to child event
`c` exists when changing only `p`'s bit changes `c` under the local rule at
the actual neighborhood state. These Boolean-derivative edges produce an
actual-sensitivity graph for the observed trajectory.

The graph's layer labels are implementation coordinates, not a metric time
input to the finality profile. Causal order comes from reachability.

## Observer Access

An observer is a bounded set of terminal cells. Its accessible record of a
seed is the counterfactual trace restricted to that set.

The laboratory evaluates:

- a global observer with access to the full terminal slice;
- translated local-window observers with equal access width.

Observer dependence is therefore tested by changing access only, not by
changing the underlying dynamics.

## Emergent Finality Profile

For seed `s`, observer window `O`, and a fixed number of causal layers:

```text
E_O(s) = (
  accessible_support,
  spatial_redundancy,
  independent_terminal_branches,
  terminal_intervention_cost
)
```

- accessible support: terminal cells changed by flipping `s`;
- spatial redundancy: distinct accessible terminal cells carrying the trace;
- independent terminal branches: connected components of trace cells when
  adjacency is inherited from the periodic lattice;
- terminal intervention cost: Hamming changes required to make the observed
  counterfactual terminal state equal to the baseline terminal state.

The first, second, and fourth quantities coincide for a raw binary terminal
trace. This is itself an important result: the richer T1 preorder does not
emerge automatically from the simplest dynamics. Branch structure can still
diverge.

## Reversal Measures

The lab reports four non-equivalent quantities.

### Causal Trace Cost

The number of observer-accessible terminal cells carrying a seed trace.

### Graph Intervention Cost

The minimum terminal-cell interventions needed to erase that trace for the
observer. In the first model this equals trace Hamming weight.

### Computational Preimage Cost

For a finite transition map, enumerate candidate predecessor states in
lexicographic order. Report:

- preimage count;
- candidates examined before all preimages are known;
- whether the predecessor is unique.

This is an exact finite benchmark, not a claim of asymptotic computational
irreducibility.

### Thermodynamic Information-Loss Bound

For a deterministic map `Y = f(X)` with uniform input over the complete finite
state space:

```text
lost_bits = H(X | Y) = H(X) - H(Y)
minimum_heat = k_B T ln(2) * lost_bits
```

The physical reference model is an array of degenerate binary memories in
contact with a heat bath at `T = 300 K`, reset quasistatically. The value is a
Landauer lower bound for logically erased information under the stated
uniform input distribution. It is not the actual dissipation of a physical CA
device.

## Required Counterexamples

The laboratory must search for and report:

1. a reversible trajectory with a nonzero observer-accessible record trace
   but zero transition-map information loss;
2. an irreversible transition map with positive information loss but a seed
   whose record is inaccessible to at least one bounded observer;
3. two equal-size observer windows receiving different finality profiles from
   the same dynamics;
4. a rule or trajectory where the raw profile dimensions collapse, showing
   that D1's four dimensions are not universal consequences of local update.

## Verdict Classes

- **Universal structure:** survives both reversible and irreversible
  substrates with the same formal definition.
- **Substrate-specific mechanism:** well-defined in one class of dynamics but
  absent or different in another.
- **Analogy only:** similarity cannot be represented by the same mathematical
  quantity or refinement map.
