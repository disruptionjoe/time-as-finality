# Saunders / Block-Universe Absorber Audit

Status: repo-local analysis. No claim promotion.

Date: 2026-07-02.

Purpose: preserve the useful part of the Saunders comparison as an absorber
screen for Time as Finality, without importing Saunders, Everett, B-theory, or
block-universe language as support.

## Plain-English Synthesis

The Saunders comparison is useful because it names the strongest philosophical
absorber for this repo:

```text
Maybe finality is not new temporal structure.
Maybe it is only observer-perspectival bookkeeping inside a fixed relational
block.
```

That is the right pressure test. Time as Finality should not defend against it
rhetorically. It should ask what, if anything, remains after the block-friendly
neighbor is granted all legitimate structure:

- fixed 4D event relations;
- worldlines / observer histories;
- causal order and domains of dependence;
- local records along worldlines;
- decoherence / Everett-style branch-relative records where relevant;
- ordinary resource, transition-system, provenance, and boundary completions.

If those completions explain the candidate finality distinction, the result is
translation residue or heuristic residue, not a physics upgrade.

## What The Outside Read Gets Right

- TaF is closer to a block-compatible / B-theory reading than an issuance or
  active-becoming theory. It does not need time to be produced.
- The repo's strongest present role is explaining why embedded observers see a
  directed past: records, access boundaries, and operation availability make
  some relations harder to revise from inside a bounded perspective.
- Partial orders and incomparability fit the relativity-friendly constraint:
  no global scalar present or universal total order should be assumed.
- The repo's demotion discipline is compatible with Saunders-style pressure:
  apparent passage should be explained, not promoted to objective becoming.

## What The Outside Read Overstates

The following readings are not currently earned as strong repo claims:

- "Colimits yield a consistent global order" as a general result.
- "Directionality emerges physically from finality gradients" as an
  unqualified arrow-of-time claim.
- "Decoherence alone is not enough for finality" as a broad measurement
  theorem.
- "Finality adds ontological weight beyond the block" without a typed residue
  surviving mature absorbers.

Recent artifacts repeatedly narrow these claims. The live pattern is:

```text
candidate finality/capability distinction
  -> add the missing boundary/resource/substrate/transition data
  -> distinction often factors through the completed neighbor state
```

That is useful progress, but it is not claim promotion.

## The Block Absorber

Before future TaF work uses Saunders-adjacent language, run this absorber:

```text
Known neighbor:
  block-compatible relational spacetime / B-theory reading, optionally with
  Everett/decoherence records when the witness is quantum.

Grant:
  all fixed event relations, worldline-local records, causal domains,
  observer access profiles, branch-relative records, resource states,
  transition relations, provenance, and boundary completions normally allowed
  by the neighboring account.

Question:
  Does the proposed TaF capability/finality distinction still fail to factor
  through that completed structure?
```

If no, record absorption.

If yes, the residue must be typed as a narrow object, not as "time really
passes" or "the block is incomplete."

## Separation Of Claims

Future writeups should keep three layers separate:

1. **Block-compatible interpretation.** Finality may explain experienced
   pastness/passage from inside a fixed relational structure.
2. **Formal reconstruction.** Record/access data may reconstruct a partial
   order in a declared finite or mathematical setting.
3. **Physical arrow / ontology.** Finality may add a physical arrow or
   ontological structure beyond established physics.

Layer 1 can be useful as interpretation.

Layer 2 requires the normal executable/theorem discipline.

Layer 3 is currently heavily absorbed and should not be stated without a
runnable artifact that survives the block absorber plus physics-native
absorbers.

## Gaps To Preserve

### 1. Static Block Reconciliation

Open question:

```text
Can every TaF finality gradient be represented as a fixed relation among
events, records, worldline-local access, and operation availability in a
static block?
```

If yes, TaF is a disciplined internal-observer description, not new temporal
ontology.

### 2. Passage Illusion Mechanism

Open question:

```text
Does finality give a precise mechanism for the experienced direction of time
without asserting objective passage?
```

This is likely a good explanatory target, but it is not automatically a
physics claim.

### 3. Everett / Branching Absorber

Open question:

```text
Are TaF's quantum-record distinctions already expressible as branch-relative
records plus observer access in Everett/decoherence-adjacent accounts?
```

The repo has Q1A/Q1C neighbor material, but a Saunders-style branching
multiplicity pass would be a useful focused absorber before any quantum-facing
claim revival.

### 4. Persistence And Multiplicity Of Selves

Open question:

```text
Can D2 observer roles and record-finality profiles say anything precise about
persistence of agents across worldline stages, or does existing block-worldline
structure already own the whole account?
```

Treat this as an optional philosophy-facing audit, not a core route unless it
produces a typed finite witness.
