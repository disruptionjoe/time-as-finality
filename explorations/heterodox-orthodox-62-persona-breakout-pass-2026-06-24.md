# Heterodox vs Orthodox 62-Persona Breakout Pass

## Status

Exploratory persona-panel artifact. Not a claim update, test result, roadmap
commitment, or evidence that the North Star is true. Its purpose is to break the
repo out of a local minimum, then hand the breakout to a falsifiable test.

## Method

A successor to the 2026-06-21 62-persona pass, run with a sharper instrument:

1. Split the 62 canonical personas into two camps: **orthodox** (31) and
   **heterodox** (31).
2. Each camp produces a one-line position per persona, then a single strongest
   **steelman**.
3. Run a **Hegelian dialectic inside each steelman** (thesis -> the camp's own
   sharpest internal antithesis -> synthesis), so each camp self-critiques
   rather than only arguing across the aisle.
4. **Meta-synthesis across the two camp syntheses**, aimed explicitly at the
   direction neither camp's local optimum can see from inside itself.

Efficiency note: the shared frontier context was read once in the main thread and
distilled into a single brief handed to both camp-workers, so no worker re-read
the repository.

## The Local Minimum Being Targeted

Every cycle the program asks an **object-level** question (is this candidate a new
mathematical object?) and the answer is always no, because a mature neighbor
absorbs it: CSP, why-not provenance, abstract interpretation, lenses/optics,
resource theory, causal inference, database view-determinacy, gauge frustration,
quantum Darwinism / SBS. T220 took the last standing candidate (LossKernel / TF1)
and factored its witness obligation through the neighbor-visible map
(`obligation = psi . nu`), collapsing it to a canonical normal form. The neutral
paper audit returned independent motivation **NOT EARNED**. The program becomes
more rigorous and less independently motivated each pass. The 2026-06-21 pass
already produced the orthodox resolution ("typed observer-shadow audit over known
math; build a Shadow Atlas; absorbers are gates"); the repo is stuck in that
optimum where rigor equals deflation.

## Cross-Camp Meta-Synthesis (The Breakout)

Both camps, from opposite premises, converged on the same escape. The local
minimum exists because every cycle asks an **object-level** question and every
object is absorbed by some host. The escape is to change the unit of analysis from
the object to the **map between absorbers**, which the per-domain absorber
discipline is structurally blind to.

The two camps named the two faces of that map:

- **Orthodox face, the predictive absorption functor:** `(residue + active
  AC-conditions) -> (absorbing neighbor + canonical collapse-witness)`. Its
  invariants (which AC-signature forces which host) are real mathematics no single
  absorber states. The win condition is that the catalogue must *predict* which
  host fires, not record it after the fact.
- **Heterodox face, the transport law kappa:** `kappa(nu)` = rank of global
  sections indistinguishable under the neighbor-visible map. Compute in domain A,
  transport via a structure-preserving map, predict domain B before native
  measurement.

These are dual: orthodox names the *domain* of the map (which host absorbs),
heterodox names the *value* it carries across (kappa). Together: **a predictive
absorption functor whose action on a residue's AC-signature both selects the
absorbing host and transports a quantitative invariant kappa that must match the
host's native obstruction value.**

### Cross-camp Hegelian

**Thesis (heterodox):** the cross-domain recurrence is a real substrate-independent
invariant the absorbers hide. **Antithesis (orthodox + heterodox's own knife):**
200+ tests, every witness absorbed, none ever survived; the recurrence is
consistent with projecting one template onto math that merely rhymes; a pattern
that fits every domain and forbids no observation explains nothing. **Synthesis:**
stop claiming the recurrence is present (vacuous) and force it to be predictive
between charts. The invariant is real **iff transportable**: a value fixed in A
constrains, in advance, the measurement in unrelated B with no shared derivation.
This reframes the program from "find a non-absorbable object" (loses every cycle)
to "find a transport law between absorbers" (which no single absorber can own, and
which 200 absorption events have never once tested for).

### Why this escapes the local minimum

The deflation engine is per-domain by construction, so it can neither generate nor
refute a claim ranging *over* domains; it is blind to exactly this object.
"Everything gets absorbed" flips from defeat into the **experimental apparatus**:
each absorption event becomes a measurement of the transport map. And it is
falsifiable, with a built-in kill switch:

- **Pass** (kappa transported from A predicts B's native obstruction across two
  unrelated absorbers, no shared derivation): a real cross-domain classification
  theorem living in no single absorber. This is the independent motivation the
  paper audit said was not yet earned.
- **Fail** (kappa must be re-tuned per domain): the recurrence was a projected
  template, the deflation verdict stands, and the North Star's central bet is
  cleanly killed instead of slowly starved.

Either outcome is non-vacuous. That is the exit from "rigor equals deflation."
