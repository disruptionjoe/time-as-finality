## What arrived

arXiv **2602.10695** (Yamaguchi, Rullkotter, Shehzad, Wagner, Tutschku, Kempf),
"Experimental demonstration that qubits can be cloned at will, if encrypted with a
single-use decryption key." Any number of **encrypted** clones by a unitary; each
decryptable by a unitary; **decryption consumes a single-use key**, so exactly one
clone is ever authoritative. Authors frame it as **in agreement with no-cloning**,
not a violation. Companion theory arXiv 2501.02757; secret-sharing adjacent arXiv
2605.26866.

## Why it is captured

It matches lane B's recorded wake condition precisely: *"a different conserved /
priced quantity."* The single-use key is a consumable resource external to the
unreadable band, and "readable copies + remaining key-authority = 1" is conserved by
construction, unlike the falsified `J + R`. Structurally: encrypted clone =
un-finalized record, decryption = finalization event, single-use key = finalization
achievable exactly once.
