## T12 / T90 Audit

### Original T12 ambition

T12 aimed to turn TaF into testable physics by finding a weak-measurement or
continuous-monitoring setup where TaF finalization time differs from standard
decoherence time while using measurable D1 coordinates.

### What T88 / T90 changed

The repo now treats that ambition as valid only if a pre-registered,
independent D1 observable changes the TaF verdict while standard monitored
statistics stay fixed. Pure threshold changes on coherence loss, fragment
redundancy, or access do not count.

### What T91 / T93 / T94 changed

- The named superconducting homodyne, uncollapse, and quantum-jump-reversal
  routes fail the independent-axis gate.
- Control-pulse energy, schedule bookkeeping, and success-conditioned reversal
  do not count as independent undo-cost evidence.
- Weak measurement is therefore demoted below detector provenance until a real
  platform names a verdict-changing independent axis.

### Consequence

T12 is currently an obstruction program, not an active physics prediction.
T90's present value is discipline: it prevents thresholded decoherence or
trajectory bookkeeping from being relabeled as a TaF discriminator.

## Result

The current Q1/T12/T90 branch does not earn a nonstandard quantum prediction.
Its surviving content is narrower and should be described in three layers:

1. Q1 is a roadmap umbrella, not one paper-facing `partially_supported` claim.
2. Q1A is observer-indexed access/provenance bookkeeping over already formed
   records.
3. Q1C is a dormant gate saying weak measurement matters only if an independent
   branch/provenance/undo-cost observable is named before modeling.

The one genuinely useful organizing move that survives this audit is the repo's
insistence on separating:

- decoherence;
- environmental redundancy;
- observer access;
- provenance / independence partitioning;
- later certification or admissibility.

That is valuable discipline, but it is not yet new quantum dynamics, not yet a
new interpretation with a forced external distinction, and not yet an empirical
discriminator.

## Blocker

No current fixed-data witness survives once provenance-aware fragment
partitioning is shared with the nearest neighbor, and no current
weak-measurement platform supplies the pre-registered independent axis required
by T90 and T93.

Pick one of two hard routes and stop mixing them:

1. Q1A external-distinction route: construct or kill one fixed-data witness
   where branch support or reversal-cost content survives after provenance-aware
   access support is matched.
2. Q1C physics route: name one concrete monitored platform with a calibrated,
   pre-registered independent axis that changes a TaF verdict while coherence,
   redundancy, access, and reversal-success summaries stay fixed.

If neither route can be specified concretely, the honest paper-facing posture
is to present this branch as interpretation discipline and record admissibility
