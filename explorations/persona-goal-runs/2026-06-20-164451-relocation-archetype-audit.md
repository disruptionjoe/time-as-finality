# Relocation Archetype Audit

- timestamp: 2026-06-20T16:44:51-05:00
- status: exploratory audit
- goal: Test whether the six GU-inspired relocation archetypes are merely metaphors or repeatedly useful classifications when structure is projected away.
- scope caveat: This is not a physics claim, not a GU claim, and not a redefinition of charge, spin, mass, phase, gauge, or Lorentz structure.
- method: Repo-context read plus four delegated subagent passes over internal LossKernel evidence, engineered systems, biological/social/cognitive systems, and hostile counterexamples.

## Executive Verdict

The six archetypes are useful as a first-pass audit vocabulary, but they do not survive pressure as six independent foundational classes.

The stronger result is a smaller operational core:

| Fiber pattern | Operational relocation | Relation to six labels |
| --- | --- | --- |
| Mixed source-lift verdicts | reconstruction debt / admissibility debt | charge-like, spin-like, mass-like, and phase-like are often variants |
| Uniform forbidden source lifts | stable hard constraint / access boundary | Lorentz-like and some charge-like cases |
| Uniform invariant source lifts | absorbed freedom / no operational debt | gauge-like control |
| Hidden variation filled by model, norm, or authority | imposed substitute rather than recovered witness | missing archetypes needed |
| Incommensurable target medium | no single reconstruction target exists | missing archetypes needed |

The six labels repeatedly help users ask the right question:

```text
What target-side operation still depends on the structure projected away?
```

But the labels are misleading if treated as physics-derived kinds. They should be neutralized before being promoted:

| Current label | Safer neutral label |
| --- | --- |
| Charge-like | admissibility / allowed-action debt |
| Spin-like | orientation / symmetry-coordinate debt |
| Mass-like | persistence / identity-inertia debt |
| Phase-like | history / path-memory debt |
| Gauge-like | bookkeeping freedom / invariant absorption |
| Lorentz-like | access-frame / observer-availability constraint |

## Missing Archetypes

The audit found seven recurring cases not cleanly covered by the six:

| Missing archetype | Definition | Examples |
| --- | --- | --- |
| Task-irrelevant absorption | Hidden variation exists but no target task changes | Gauge representative, perceptually invisible compression |
| Adversarial recontextualization | Loss changes what another agent can exploit, infer, spoof, or hide | Redaction, security logs, detector spoofing |
| Normative reassignment | A rule, authority, liability regime, or institutional convention assigns consequences after loss | Legal outsourcing, governance delegation |
| Invariant-selection tradeoff | Projection preserves one invariant by sacrificing another | Map projections, schema normalization, compression settings |
| Sidecar externalization | Lost structure is moved into metadata, logs, source maps, signatures, or audit trails | Compiler source maps, raw-log contracts, Git/PR metadata |
| Incommensurable value loss | The target medium changes what counts as successful preservation | Poetry translation, qualitative research summaries |
| Prior substitution | A model fills missing structure with learned/default priors rather than source-derived witnesses | ML embeddings, risk scores, cognitive stereotypes |

Some of these may reduce to the three fiber outcomes if formalized carefully. But treating them as named hostile categories prevents the six labels from overfitting social and cognitive domains.

## Collapse Analysis

The six proposed archetypes are not independent under the T107 rule.

Charge-like, spin-like, mass-like, and phase-like mostly collapse to:

```text
mixed source-lift verdicts -> reconstruction/admissibility debt
```

Lorentz-like usually collapses to:

```text
uniform forbidden lifts -> stable access constraint
```

Gauge-like is special because it is the cleanest negative control:

```text
uniform invariant lifts -> absorbed freedom
```

This means the six labels are better used as domain-facing descriptors layered over a smaller operational taxonomy.

## Recommended Revisions

1. Classify by fiber outcome first.
2. Use neutral labels instead of physics terms in general-domain reports.
3. Preserve the six physics-inspired labels only as GU-crosswalk mnemonics, never as ontological claims.
4. Add hostile classes for adversarial, normative, invariant-tradeoff, sidecar, value-loss, and prior-substitution cases.
5. Require every positive case to state:
   - what source structure was projected away;
   - what target judgment or operation remains;
   - whether source lifts are mixed, uniformly forbidden, or uniformly invariant;
   - what reconstruction debt, constraint, or absorption follows;
   - whether ordinary provenance/effects/abstract interpretation/lenses/CSP already explain the case.
6. Treat `gauge-like absorbed freedom` as a required negative control in every future audit.
7. Treat T108 as binding: no novelty claim until a same-neighbor-data quotient fixture survives.

## Assessment

| Criterion | Verdict | Reason |
| --- | --- | --- |
| Useful | Yes | The archetypes help scan diverse projection/loss phenomena and ask what operational consequence remains. |
| Redundant | Partly | T108 shows the formal behavior is currently absorbed by existing provenance, abstraction, lens, CSP, and effect machinery. |
| Misleading | Potentially | Physics labels imply more structure than the audit establishes. Neutral labels are safer. |
| Potentially foundational | Not yet | The foundational candidate is not the six labels; it is the smaller source-fiber pattern plus reconstruction-debt accounting. |

Best current classification:

```text
useful but non-foundational audit vocabulary
```

The six labels survived as a heuristic. They did not survive as six independent archetypes. The pressure test strengthens the broader relocation program only if it is reframed around source-fiber outcomes, negative controls, and reconstruction debt.

```text
projected source structure
-> source-fiber outcome
-> reconstruction debt / stable constraint / absorbed freedom / sidecar externalization
-> emergence consequence
```

Required positive examples:

- detector provenance raw-log admissibility;
- Git squash/history reconstruction debt;
- database view/update/provenance debt;
- consensus snapshot/finality debt;
- public infrastructure maintenance debt;

Required negative controls:

- gauge/invariant absorption;
- task-irrelevant compression;
- sidecar externalization where source maps or logs fully carry the lost structure;
- incommensurable value-loss case where reconstruction debt is the wrong model.

Promotion condition:

```text
Promote only if debt observables predict target-side operational failures better
than ordinary provenance/effect/abstract-interpretation/lens/CSP descriptions,
or if they provide a compact normal form that consistently improves audits
without claiming novelty.
```
