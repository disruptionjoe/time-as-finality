# Recurring Structure Map v0.1

## Purpose

This map asks what the repo is repeatedly discovering after removing the
dramatic language around time, consciousness, and quantum interpretation.

The current answer is:

```text
finite local-to-global obstruction under typed information loss
```

The repo should not claim that it discovered gluing obstruction. Gluing
obstruction is known from sheaf theory, contextuality, cohomology, and
constraint satisfaction. The possible original contribution is a finite typed
calculus for tracking when projection, restriction, observer access, or
morphism composition creates, hides, displaces, or reveals obstruction.

## Pattern Vocabulary

| Pattern | Meaning |
| --- | --- |
| local-to-global failure | Locally valid data do not determine a global section, completion, or coherent assignment. |
| typed forgetting | A morphism loses named structure, and that loss matters for later attribution. |
| path dependence | Same endpoints can receive different verdicts because different paths forget different structure. |
| access boundary | Records or witnesses exist, but are outside a bounded region's reachable set. |
| provenance / independence | Accessible records may be copies or dependent witnesses rather than independent support. |
| auditability gap | A system can operate or close correctly while the evidence certifying that fact lives outside it. |
| obstruction displacement | An obstruction changes layer or invariant rather than simply disappearing. |
| descent / reconstruction failure | Compatible local data are insufficient for canonical global reconstruction without identity, overlap, or compatibility data. |

## Emerging Spine

The strongest candidate spine is:

```text
typed finite restriction systems
  + information-losing morphisms
  + first-class loss kernels
  + admissible obstruction attribution
```

Access/provenance is the observer-facing and physics-facing manifestation.
Typed forgetting is the more general mathematical structure.
