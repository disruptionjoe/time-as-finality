# N7: Q1A Measurement-Neighbor Positioning

Status: primary sources checked 2026-06-21.

This note positions Q1A against the nearest quantum-measurement and
interpretation neighbors after T102, T104, T105, T109, and T118. The purpose
is not to defend Q1A. It is to state, with explicit prior-art ownership, what
Q1A would still need to add to become more than bookkeeping.

## Current Repo Verdict

The present strongest Q1A claim is narrow:

```text
do not treat raw fragment redundancy as observer-relative classical finality
until the access cut and provenance-aware independence partition are physically
audited.
```

That is an admissibility rule over already formed records. It is not a new
measurement dynamics, not a collapse rule, not a Born-rule derivation, and not
currently a distinct interpretation.

## What Each Neighbor Forces

### 1. Decoherence

Zurek's decoherence program already owns the main dynamical claim that
environmental monitoring suppresses interference and selects stable pointer
states. Q1A does not currently compete with that.

T2 and T62 support only the weaker sentence:

```text
the same decohered record structure need not count as finalized for every
observer if the observer's physical access cut differs.
```

That is not yet a new measurement theory. It is a reminder that
observer-relative admissibility is stricter than "decohered somewhere in the
environment."

Kill condition for any stronger Q1A reading:

```text
if every Q1A witness can be rewritten as decoherence plus ordinary observer
reachability, Q1A has no independent measurement content against decoherence.
```

### 2. Quantum Darwinism

OPZ and Zurek already own the redundancy story. After T103, the only plausible
Q1A wedge was that two cases with the same raw fragment count could still
receive different D1 verdicts because one duplicate fragment shared provenance.

T104 and T105 remove that wedge in the present family. Once Quantum Darwinism
is allowed to count audited provenance classes instead of raw copies, the
current finalized/not-finalized split matches D1's verdict-critical content.

So the current surviving Q1A content is only:

```text
audit the fragment partition before counting redundancy as objectivity or
finality evidence.
```

That is valuable discipline, but it is not a distinct physics claim.

### 3. Consistent / Decoherent Histories

Griffiths, Gell-Mann, Hartle, and Halliwell already own the main records-of-
histories territory: probabilities for decohering histories, quasiclassical
domains, and explicit discussions of where records are stored when histories
decohere.

Against this neighbor, Q1A currently lacks a theorem saying that its
observer-indexed finality preorder classifies cases that cannot already be
handled by:

- a choice of decoherent history family;
- a records-of-histories account;
- or an IGUS-relative quasiclassical description.

Until that exists, Q1A should not be described as a new records framework.

### 4. Relational Quantum Mechanics

Rovelli already owns the core move that facts are relative to interactions and
systems rather than absolute observer-independent state assignments.

For Q1A to survive this comparison, it would need physically load-bearing rules
for:

- which relative records count as independently supported;
- when access is too incomplete for finality attribution;
- and why those thresholds are not just optional bookkeeping conventions.

The current repo does not earn that. T102 correctly treated RQM as a major
absorption threat, and nothing after T102 reverses that.

### 5. QBism

QBism owns agent-centered probability assignments and treats the Born rule as a
normative constraint on expectations rather than a report of observer-
independent quantum state. This means Q1A should not present "observer-relative"
language as if that alone were new.

Q1A does remain categorically different in one respect:

```text
it talks about physical record admissibility, not only personal credence.
```

But that is only a category split. It is not a positive wedge until Q1A names a
physical record predicate with non-absorbed verdict content.

### 6. Everett / Many-Worlds

Everett already owns branch-relative observer states inside unitary dynamics.
Q1A therefore does not earn distinctness merely by saying that records can be
branch-relative, observer-indexed, or available only after later comparison.

The obvious rescue routes inside the repo were branch support and reversal cost.
T109 and T118 close those routes for the current fixed-data family. So there is
no live Everett-facing wedge at present beyond audited record bookkeeping.

## Net Verdict

The honest external positioning after T104, T105, T109, and T118 is:

```text
Q1A is not currently a distinct quantum measurement framework.
It is an admissibility and bookkeeping discipline over already formed records.
Its strongest content is negative: raw fragment count should not be promoted to
observer-relative classical finality until access and provenance are audited.
```

This is still useful. It is not Nobel-level by itself, but it is the sort of
narrowing the repo needs before any broader quantum claim becomes credible.

## What Would Reopen Q1A

Q1A regains external interest only if one of two things happens.

1. A fixed-data witness survives after all of the following are shared with the
   neighbor:
   - decoherence / pointer-basis evidence;
   - fragment-information summaries;
   - ordinary branch/history availability;
   - provenance-aware fragment partition.
2. Q1A derives its admissible partition or threshold rule from extra physical
   structure that the neighboring framework does not already own.

After T105, T109, and T118, that witness must also do more than restate:

- audited accessible provenance-class count;
- branch-history variation;
- or hidden undo/erasure bookkeeping inside the same family.
