## Reference Scenario

The graph contains two causally independent record branches, `A` and `B`,
which meet before a later record branch `C`. Every proposition has two
supporting records and a reconstruction threshold of two.

Reconstructed relation:

```text
A < C
B < C
A || B
```

`A || B` means the formalism preserves incomparability. The result is
unchanged across all topological traversals of the finite graph.

## Access Loss Versus Reversal

Removing holder `ha2` from observer `O`'s access set reduces `O`'s accessible
support for `A` from two records to one, while both global records remain
active.

Erasing record `rb2` reduces active support for `B` from two to one for every
observer that could inspect that holder.

The model therefore represents local access loss and physical record erasure
as distinct operations.

## Minimal Counterexample

The smallest graph that can present two independent proposition records to
one reconciler has:

- one formation event for `L`;
- one causally independent formation event for `R`;
- one observation event in their common causal future.

Both propositions reach threshold one, but the reconstructed relation is
empty. Any total order between them would have to be imposed by a traversal,
clock, tie-break rule, or observer processing sequence not present in the
causal record structure.

This refutes the strong claim:

> Accessible finalized records always determine a total temporal order.

It does not refute the weaker claim:

> Accessible record-stabilization frontiers can reconstruct an
> observer-relative temporal partial order.
