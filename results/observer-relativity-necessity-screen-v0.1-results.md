## Plain-English Reading

T381 says:

```text
If we require the ingredients needed for observer relativity, the adapter wants
to be exactly two independent primitive null directions.
```

The big improvement is that the two-null-channel object is no longer just
"a convenient choice." It is the only minimal survivor in this finite screen.

The big limitation is that the screen itself supplies the requirements:

```text
source-compatible counts
nullness
bilinear interval
minimality
```

So this does not yet derive the adapter from compatibility alone.
