# Quantum Measurement Finality Results

Result: T2 focused tests pass `7/7`.

## Model

T2 v0.1 uses five qubits:

```text
S, A, E1, E2, E3
```

The pointer basis is:

```text
computational_z
```

The interaction chain is:

```text
CNOT(S -> A)
CNOT(A -> E1)
CNOT(A -> E2)
CNOT(A -> E3)
```

The model uses reversible unitary interactions only. It never selects a
collapse outcome.

| outside observer | none | `(0, 0, 0, 0)` |

## Main Finding

The final state is decohered in the pointer basis:

```text
pointer coherence = 0.0
environment R_delta = 3
```

But the outside observer has:

```text
D1 = (0, 0, 0, 0)
```

So T2 detects a decohered-but-inaccessible case. This supports the narrow
claim that "decohered globally" and "final for this observer" are distinct
predicates in the toy substrate.

## D1 Boundary

Holder redundancy and accessible support are generated by the dynamics. But
branch support remains `1` because every record descends from the same pointer
measurement root, and reversal cost is only an inverse-operation proxy.

## Claim Recommendation

Keep Q1 `partially_supported`. T2 v0.1 shows that D1 adds an observer-access
distinction beyond decoherence and raw `R_delta`, but it does not solve the
measurement problem or physically settle every D1 dimension.
