# T16 Dynamical Phase-Bearing Record Results

Result: T16 focused tests pass `6/6`; the current full branch suite passes
`91/91` after T17 was added.

## Derivation Rule

T16 derives signed record weights from local dynamics:

```text
terminal 0 -> 1 trace cell gets weight +1
terminal 1 -> 0 trace cell gets weight -1
```

The trace itself is generated by changing one seed bit in an elementary
cellular automaton and comparing the terminal state with the unperturbed
terminal state.

## Sweep

The deterministic sweep evaluates:

```text
256 rules * 3 layer depths * 16 width-4 initial states * 4 seed positions
= 49,152 cases
```

| Metric | Value |
| --- | ---: |
| nonempty trace cases | `39,488` |
| mixed-sign trace fraction | `0.2504` |
| zero-readout fraction | `0.1823` |
| phase-blind profile count | `5` |
| profiles with multiple readouts | `4` |
| multiple-readout profile fraction | `0.8000` |

## Profile/Readout Separation Witness

T16 finds two dynamically generated traces with the same phase-blind profile:

```text
(2, 2, 1, 2)
```

Left witness:

```text
rule: 2
initial: 0001
seed: 2
weights: +1, -1
readout: 0.0
```

Right witness:

```text
rule: 3
initial: 0000
seed: 0
weights: -1, -1
readout: 4.0
```

The finality profile is the same. The signed readout differs.

## Cancellation Trajectory

T16 also finds a generated trajectory:

```text
rule: 6
initial: 0001
seed: 1
```

Readout by layer:

```text
0.0, 1.0, 0.0, 1.0, 0.0, 1.0
```

The zero values come from generated mixed-sign traces such as:

```text
+1, -1
```

## Verdict

T16 removes the largest weakness of T13-T15 in a toy setting: signed weights
are no longer assigned by hand. They are derived from actual local trace
direction.

The limit remains important. This is not a quantum derivation. It is a finite
classical local-dynamics witness that phase-like signed readout can be
generated by a substrate while the finality profile remains phase-blind.
