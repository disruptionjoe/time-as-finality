   Reframed T411 as U(1)-asymmetry / athermality resource theory under a

   A activates at R+, B free at R+ — a TOTAL chain `B =< A`. The two
   residues T411 CLAIMS both absorb: "work cannot buy reach" is merely the
   free-ancilla monotonicity axiom (proved by the artifact's own dilation
   lemma), and `beta = 0` non-locality is ordinary global-monotone
   marginal-blindness. Computed: reach R -> both free (locked-vis 0.0);
   R+ -> A 0.989743, B 0.0; appending pure + Gibbs work ancillas leaves
   A unchanged to delta 0.0. **Sole survivor:** the assertion that the
   R -> R+ cut is PHYSICALLY FORCED — which lies outside resource
   theory's vocabulary and is exactly what the artifact concedes is
   adopted, not derived. Net: the finite-witness formal content collapses
   into standard asymmetry/athermality resource theory; the survivor is a
   physicality assumption already flagged unproven.

2. **Joint-record completion (the T401 move) — ABSORBED.** State A retains
   tier-1 and state B retains tier-1 (only tier-2 departs), so in BOTH
   runs the tier-1 carriers are physically CO-PRESENT. R is a declared
   strict subset of the retained register set; excluding retained tier-1
   from R is pure declaration, not a physical departure. Computed: in A
   the retained tier-1 carries `I(R:E1) = 1.97` bits (`beta = inf`), and
   the leg-1/2 R-equality is literally the R+ state's marginal after
   discarding this retained correlated register (R-marginal == R-run to
   0.0); a single Z-measurement on the retained `E1_1` separates A from B
   (`P = 0.4286` vs `0.0`); at `beta = 0` the full retained R+ joint
   distinguishes A from B at TD = 0.4949. The separating boundary R -> R+
   crosses only declared bookkeeping; the one genuinely physical departure
   (tier-2, R+ -> R++) does no discriminating work (A and B already differ
   at R+ before any tier-2 access) and drives only the priced enlargement
   control T411 itself concedes is standard (T399). This is the enlarged-
   state / joint-record completion move the program forbids. A kill on the
   discriminator claim. Note: the artifact's honest leg
   `boundary_physicality_reduces_to_declaration` did not fire because it
   only audits whether SEPARATION verdicts at R+ are certificate-carried

model-class level. The physical boundary is **not earned**; T407's
declared-vs-physical objection stands undischarged, consistent with the
artifacts' own conditional concession ("adopted, not derived").

genuinely fails — the discriminator's datum is a global correlation in no

cone, so the content is not a causal-reachability relabeling; (b) the
resource-theory absorber leaves only the physical-forcing assertion
un-absorbed — the very thing the joint-record and decoherence moves kill;
(c) the re-scoped standing stands intact: repo-internal calibration of the
certificate toolkit (all-channel certificates, graded bands, T142 ledger)
