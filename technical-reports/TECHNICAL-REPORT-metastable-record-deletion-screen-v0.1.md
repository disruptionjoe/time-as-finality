# Technical Report: Metastable-Record Deletion Screen v0.1

## Claim Under Test

T145 and T148 leave H7 with one plausible physical-arrow reinstatement route:
a physically typed record substrate where physical deletion remains
constructor-impossible after full absorber accounting.

T152 tests the obvious substrate candidate: metastable records protected by
finite barriers.

## Result

No tested finite metastable fixture clears H7.

Finite barriers create retention lifetimes and task capabilities, but deletion
remains finite-rate or finite-control possible. Barrier height, reservoir
state, control windows, and task horizon are ordinary kinetic/resource
variables. If deletion is blocked by an infinite barrier, the result comes from
an ideal constructor restriction or state-space split. If deletion is blocked
only for an observer lacking control, the result is boundary/access accounting.

## Current Strongest Claim

Metastability does not reopen H7 as a thermodynamic-arrow claim. Finite
metastable barriers can support useful record retention and future-operation
capabilities, but they do not make deletion constructor-impossible. The only
matched finite-barrier split found here is the already-demoted T145
future-operation/topology residue.

## What This Improved

T152 converts "stable physical record" into a declared barrier, reservoir,
control, horizon, and provenance audit. It blocks a likely overclaim before it
enters paper-facing H7 prose.

## What This Weakened

The remaining physical-deletion upgrade path is narrower. A future H7 witness
cannot cite metastability alone; it must show finite-substrate impossibility
after the metastable absorber vector is matched.

## Falsification Condition

T152 fails in H7's favor only if a finite, physically typed metastable record
substrate has matched barrier, reservoir, erasure, capacity, sink,
observer-boundary, provenance, source-copy, and reversible-control data, a
task-natural future-operation split, and a physical deletion reverse that is
constructor-impossible rather than merely exponentially unlikely or
observer-inaccessible.
