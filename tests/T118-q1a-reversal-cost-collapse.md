# T118: Q1A Reversal-Cost Collapse

## Route

Quantum measurement / classical records.

## Question

Does reversal cost provide any real escape from the T105 accessible-class
reduction and the T109 branch-support collapse inside the current fixed-data
Q1A witness family?

## Motivation

After T105 and T109, reversal cost is the last unused D1 dimension that could
still be cited as hidden structure inside the present Q1A branch. That is the
highest-value remaining audit because if reversal cost also collapses, the
repo should stop implying that the current fixed-data family contains any
independent measurement-theory content beyond audited record accounting.

## Success Criteria

- Audit the current T105 visible fixed-data family without changing the
  ordinary quantum-side summaries.
- State the strongest admissible reversal-cost proxy available inside that
  fixed-data gate.
- Check whether the proxy ever splits cases with the same audited accessible
  provenance-support count.
- Reuse the T103 branch-history control to verify that richer undo language is
  not being smuggled in by changing ordinary branch/history data.
- State clearly whether reversal cost is still a live Q1A rescue route.

## Failure Criteria

- The audit quietly imports control-pulse energy, Hamiltonian dynamics,
  thermodynamic work, or postselected undo data that are absent from the
  current fixed-data witness language.
- It treats an arbitrary monotone relabeling of accessible support as new
  reversal-cost structure.
- It advertises the result as new measurement dynamics rather than a negative
  structural audit.

## Claim Impact

If the only admissible reversal-cost proxy is identical to the audited
accessible provenance-support count, then the current Q1A branch has no live
independent D1 dimension left inside the fixed-data family. Paper-facing
language should then treat Q1A as record-accounting discipline unless a new
dynamic witness is built.
