# T140: Q1 Frontier Escape Matrix

## Route

Quantum measurement / classical records.

## Question

After Q1A, Q1B, Q1C, and Q1D have each been narrowed, what exactly would
count as a non-null next quantum move rather than another absorbed internal
witness?

## Motivation

The Q1 branch now has a failure mode of its own: because the child claims are
separately demoted, it is easy to keep adding finite witnesses that only
restate already absorbed bookkeeping. T140 turns the surviving branch gates
into an executable route-selection matrix.

## Setup

The matrix audits four branch-specific escape classes:

1. Q1A becomes live only if a fixed-data witness keeps standard quantum-side
   summaries, shared provenance partition, and audited accessible support
   fixed while a physically justified D1 dimension or nonimportable partition
   rule changes the verdict.
2. Q1B becomes evidence only if a T136/T138-valid manifest is signed before
   data, real event-level rows are published without schema drift, packet and
   authority gates pass, and the verdict survives T83.
3. Q1C becomes live only if a calibrated auxiliary axis remains
   verdict-changing after conditioning on the full event-level ordinary
   monitored record, without postselection or schedule-proxy dependence.
4. Q1D becomes more than a guardrail only if it names a nonredundant theorem
   target that preserves no-signalling and avoids hidden-variable readings.

## Success Criteria

- Current Q1A classifies as bookkeeping-only.
- Current Q1B classifies as externally blocked, not as evidence.
- Current Q1C classifies as dormant/null unless a full-record escape appears.
- Current Q1D classifies as guardrail-only.
- The result recommends against more internal Q1 toy witnesses unless an
  escape gate is explicitly targeted.

## Failure Criteria

- The matrix treats Q1B scaffolding as detector evidence without real rows.
- The matrix reopens Q1A through ordinary support-count variation.
- The matrix reopens Q1C through coarse-summary refinement.
- The matrix treats standard contextuality language as a new Q1D theorem.

## Claim Impact

Q1 remains a roadmap umbrella. No current child branch has an active internal
upgrade route. Q1B is the only non-null experimental direction, but it is
externally blocked pending a signed pre-event manifest and real event-level
packet.
