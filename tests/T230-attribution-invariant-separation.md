## Strongest positive result

A single decisive falsifier — the **three-gate absorption-escape test** — was
built and run against five candidate attribution invariants spanning the four
data strata of a typed-lossy morphism. A candidate clears route (b) ONLY if it
passes all three gates on a same-`nu` pair:

- **GATE 1 (separates):** `I(a) != I(b)` for some pair with `nu(a) == nu(b)`.
- **GATE 2 (non-absorbable):** the separation is NOT reproduced by any admissible
  enlargement `nu'` (i.e. it does not come from a source field that, once
  admitted as audit data, makes a mature neighbor separate the pair too).
- **GATE 3 (genuine invariant):** `I` is relabeling-stable (3a — constant on every
  bijection of the free-decoration alphabets, which fix the morphism) AND local
  (3b — a pure function of the case's own data, not an external registry).

The witness fiber is four cases sharing **one** `nu` signature (uniform-true
lift table, identical composite map and target fields), differing only off-`nu`
in hidden source datum and free decorations. Result, executed:

| candidate | outside canonical | separates | absorbed by nu' | relabel-stable | local | first failed gate |
|---|---|---|---|---|---|---|
| `canonical_obligation` (T220 psi.nu) | no | no | – | yes | yes | gate 1 (nu-measurable) |
| `source_reading` | yes | **yes** | **yes** | yes | yes | **gate 2 (absorbed)** |
| `free_label` | yes | yes | no | **no** | yes | gate 3a (not invariant) |
| `ambient_index` | yes | yes | no | yes | **no** | gate 3b (not local) |
| `source_fiber_cardinality` | yes | no | – | yes | yes | gate 1 (slides back inside nu) |

All four distinct gates fire; the no-go is non-trivially distributed, not a
tautology of one gate. A **non-vacuity check** is included: a synthetic genuine
separator is injected and shown to pass gates 1, 3a, 3b — proving the harness
*can* report a clear, so the no-go is a real negative.

The **honest subtlety** the test makes explicit: relabeling-stability and
locality are NOT jointly sufficient. The `source_reading` invariant is local AND
relabeling-stable AND separates a same-`nu` pair — it is a *real* separator. It
fails only at **gate 2**: admitting its source field as audit data enlarges `nu`
to `nu'`, which reproduces the separation in a mature neighbor. So the naive
two-clause route-(a) statement ("every local relabeling-stable invariant factors
through `nu`") is **false**; the correct citable bounded negative is the
three-clause form: *every local, relabeling-stable, non-absorbable invariant
factors through `nu` on the family.* Gate 2 is the load-bearing clause that
distinguishes a real separator from an independence-earning one.

## First exact obstruction / missing object

The absorption-escape **trichotomy is exhaustive over the off-`nu` strata**: a
same-`nu` separator must read stratum (ii) source, (iii) free decoration, or
(iv) ambient/extensional, and each stratum is closed under exactly one gate
(absorbed / non-invariant / non-local). The candidate that looks like it reads a
fourth, intrinsic kind of datum — `source_fiber_cardinality` — is the trap the
open-problem file warns about: the lift table IS part of `nu` (csp/provenance),
so the "outside" candidate slides back inside `nu` and fails gate 1.

The **missing object** route (b) needs is an attribution invariant reading a
datum that is simultaneously: intrinsic to the morphism, not in `nu`, not a free
decoration, not an external choice, and not admissible as neighbor audit data.
No such datum is exhibited on this finite family; whether one exists is exactly
Open Problem 11.1 left open. T230 does not close 11.1 in the affirmative; it
sharpens the audit's single-case NOT-EARNED certificate (T220) into a
trichotomy-exhaustion that pins the exact gap.

## Constructive next object

A **source-automorphism rigidity certificate**: an invariant keyed to a
source-side *symmetry class* of the morphism — e.g. whether the source fiber
admits a nontrivial self-map fixing every `nu`-visible datum — rather than to
the *value* of any source field. The structural bet: if such an
automorphism-class invariant separates a same-`nu` pair, **gate 2 cannot absorb
it** (there is no single source FIELD to admit; the datum is a symmetry-class,
not a value), and gates 3a/3b pass by construction (it is local and
relabeling-stable). This is the one untested crack in the trichotomy. Build the
source-automorphism groupoid of a typed-lossy morphism and test whether its
isomorphism class is `nu`-measurable on a same-`nu` fiber.
