  Uses ONLY the menu algebra acting on two given lines. ZERO metric / ghost /

The **adapter** is the natural transformation `η : F_TaF ⇒ F_phys`, components

**Single shared knob** (identical predicate on both readings): *is the ghost-parity

commutant) ⇒ superselection survives ⇒ FORCED. Non-central (irreducible `M`, Schur

## The toy (C² Krein fragment)

`V = ℂ²`, `K = [[0,1],[1,0]]` (`K = K†`, `K² = I`), ghost-parity `P = K`. Eigenlines
`(1,1)` K-norm `+2` (physical, P-even) and `(1,-1)` K-norm `−2` (ghost, P-odd).
Separating datum `d` = the ghost-line → physical-line move. Grading-breaking
generator `Z = diag(1,−1)` (`[Z,P] ≠ 0`, `Z·(1,−1) = (1,1)`).

- **M0 = ⟨I,P⟩** — abelian, reducible: P central, commutant dim 2, metric space
  dim 2 (indefinite survives). Both functors → **(graded, FORCED)**.
- **M_all = ⟨P,Z⟩ = M₂(ℂ)** — irreducible: P not central, commutant dim 1 (Schur),
  metric space dim 1 (unique definite). Both functors → **(ungraded, DECLARED)**.
- **Arrow M0 ⊂ M_all** — both functors send it to the SAME FORCED→DECLARED flip;
  naturality square commutes; one-way rule holds as lax functoriality.
- **M1 = ⟨I,P,U⟩** control — U grading-preserving (`[U,P]=0`, stays in ⟨I,P⟩):
  P stays central, verdict stays FORCED. Localizes that the flip requires a
  grading-BREAKING (resource-class A2) operator, not any enlargement.
