# T447: Quantum E3 Exact No-Go Big Swing

## Target Claims

Capability-boundary mode taxonomy, quantum E3 hinge, T435 admissible-menu gate,
and T436 resource-lift classifier.

## Setup

Attempt the exact structural-symmetry lane with a finite charge toy:

```text
System S charge sectors: {0, 1}
Target task: exact charge-flip unitary on S
A1: no reference resource; exact charge-conserving operations
A2: finite non-wrapping charge-ladder reference;
    exact total-charge conservation;
    exact catalytic return of the reference
```

Under exact total-charge conservation, a charge flip on `S` requires the
reference to shift by the opposite charge. If the reference must be returned
exactly, the finite reference state must be an eigenvector of the finite
non-wrapping shift with unit-modulus eigenvalue.

The finite non-wrapping shift is nilpotent. If `S^N = 0` and
`S eta = lambda eta` with `|lambda| = 1`, then
`0 = S^N eta = lambda^N eta`, so `eta = 0`. Therefore no nonzero exact catalyst
exists in this finite policy.

## Known Physics Constraints

This is a finite exact-catalytic toy, not a physical theorem. Its positive result
is only that a declared finite non-wrapping reference with exact catalytic return
cannot implement the charge-flip task under exact total-charge conservation.
Changing the resource policy can absorb or reroute the packet.
