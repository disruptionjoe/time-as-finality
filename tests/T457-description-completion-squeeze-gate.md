# T457: Description-Completion Squeeze Gate

## Target Claims

- [Region-Indexed Capability Discriminator](../open-problems/region-indexed-capability-discriminator.md)
- T454 integrated E3-region packet swing
- T455 T454 hostile-review gate
- T456 policy-invariant region theorem gate

## Setup

T456 rejects current T454/T455 for stronger Direction-A posture because the
packet still misses description nonfactorization, reference-policy invariance or
preclusion, and a policy-independent region theorem. T457 focuses on the first
blocker.

The current T454-style packet class uses a boundary-resource descriptor that
decides whether the boundary menu can revise the final verdict. If that
descriptor is admitted, then boundary-menu capability factors through admitted
description. If the descriptor is omitted, any apparent split is only
underdescription unless a future policy-independent theorem makes that omitted
field physically non-admissible.

## Success Criteria

- Current T454/T455 boundary-resource-description class is rejected because
  admitted description decides the boundary-menu capability.
- Omitting the boundary resource creates a same-description split but is rejected
  as underdescription without an independent nonadmissibility theorem.
- Description-complete, no-policy-theorem, reference-policy-fragile,
  hidden-label, and missing-control packets are rejected.
- Only a synthetic non-descriptive theorem target is admitted, with no claim
  promotion.

## Failure Criteria

The gate fails if it:

- treats T454/T455 as admitted for stronger Direction-A posture;
- treats underdescription as description nonfactorization;
- treats the synthetic future target as a theorem already earned;
- ignores reference-policy fragility or hidden-label shortcuts;
- edits `CLAIM-LEDGER.md`, `TESTS.md`, `ROADMAP.md`, README, North Star files,
  public posture, hard policy, or cross-repo truth.

## Known Physics Constraints

T457 is a repo-local admission/guardrail gate over finite toy packet classes. It
is not a WAY theorem, not a quantum physics claim, not a real substrate law, not
cross-repo support, and not claim-ledger movement.

## Contribution Needed

Use T457 to stop rebuilding the current T454 boundary-resource-description class.
Future Direction-A packets must change class by supplying a policy-independent
nonadmissibility theorem for any omitted capability-deciding field. If no such
packet can be supplied, demote the integrated E3-region route to a useful
negative guardrail.
