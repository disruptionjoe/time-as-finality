# T460: Post-T459 Direction-A Restart Router

## Target Claims

- [Region-Indexed Capability Discriminator](../open-problems/region-indexed-capability-discriminator.md)
- T457 description-completion squeeze gate
- T458 reference-policy invariance gate
- T459 policy-independent theorem-supply gate

## Setup

T459 demotes the current integrated E3-region packet class to a route-level
negative guardrail. The next risk is repeating that class by small variation or
misrouting adjacent E1/E2 work as if it restarted Direction A.

T460 makes the post-T459 disposition executable. A future Direction-A restart
must declare a genuinely new region packet class, supply a predeclared
independent theorem, target the named completion that would otherwise restore
factorization, make that completion physically non-admissible, and clear T457,
T458, and T459 together.

## Success Criteria

- The current T454-T459 integrated E3-region route is closed as a Direction-A
  restart candidate.
- Minor variants of that packet class are rejected.
- Theorem-only, no-theorem, post-hoc theorem, description-factorizing,
  reference-policy fragile, untargeted theorem, missing-control, public-posture,
  and cross-repo shortcut variants are rejected or blocked.
- Adjacent E1/E2 packets route through their existing mode gates first.
- Only a synthetic genuinely new Direction-A packet class is admitted as a
  future review target, with no claim promotion.

## Failure Criteria

The router fails if it:

- reopens the current T454-T459 packet class for stronger Direction-A posture;
- lets a minor variation count as a new packet class;
- admits a theorem not tied to the named completion;
- ignores T457, T458, or T459;
- treats E1/E2 packets as direct Direction-A restarts without their mode gates;
- edits `CLAIM-LEDGER.md`, `TESTS.md`, `ROADMAP.md`, README, North Star files,
  public posture, hard policy, or cross-repo truth.

## Known Physics Constraints

T460 is a repo-local routing/admission gate over finite toy packet classes. It
is not a WAY theorem, not a quantum physics claim, not a real substrate law, not
cross-repo support, and not claim-ledger movement.

## Contribution Needed

Use T460 before any future attempt to reopen Direction A after T459. If no new
packet class can supply a predeclared independent theorem clearing T457, T458,
and T459 together, choose a different lane instead of rebuilding the closed
integrated E3-region packet class.
