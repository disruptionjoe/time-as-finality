# T472: Observer-Shadow Index Admission Gate

## Target Claims

- `open-problems/observer-shadow-category.md`
- T470 observer-shadow composition gate
- T37/T41 typed transport category and path-dependence fixtures
- T220 LossKernel witness-obligation factorization certificate

## Question

Can T470's future-packet burden be made executable: observer-shadow packets
must declare all load-bearing indices before morphism comparison, and hidden
state completion must be separated from genuine preservation?

## Setup

T472 consumes the T470 finite checks and evaluates five proposed packets:

- endpoint-only transport comparison;
- path-indexed transport comparison;
- LossKernel neighbor-visible preservation;
- hidden-source omission;
- hidden-source completion.

Each packet names declared indices, required indices, completion kind, and the
intended route.

## Success Criteria

T472 succeeds if it:

- rejects endpoint-only transport because path and accumulated forgotten
  structure are load-bearing indices;
- admits path-indexed transport only as indexed bookkeeping;
- admits LossKernel neighbor-visible factoring as the genuine preservation
  control;
- rejects hidden-source omission before completion;
- records hidden-source completion as absorber/state-completion bookkeeping,
  not as new observer-shadow category evidence;
- leaves claim status, public posture, North Star, roadmap, README, hard
  policy, and cross-repo truth untouched.

## Failure Criteria

T472 fails if it:

- allows endpoint-only transport comparison to pass;
- treats hidden-source completion as a category-theorem result;
- collapses indexed bookkeeping and absorber completion into the same route;
- promotes D1, PO1, TF1, LossKernel, observer-shadow category, North Star
  geometry, or any physics/consciousness claim.

## Known Physics Constraints

None. T472 is a finite bookkeeping/admission gate. It does not make physics,
consciousness, or observer-metaphysics claims.

## Result

Implemented as T472 v0.1.

Verdict:

```text
OBSERVER_SHADOW_INDEX_GATE_BUILT_COMPLETION_SEPARATED_NO_PROMOTION
```

The gate admits a genuine preservation control and an indexed bookkeeping
completion, rejects omitted load-bearing indices, and records hidden-source
completion as absorber state completion. It supports an index-admission
guardrail only, not a global observer-shadow category theorem.
