# T506: Boundary-Adapter Dimension-Carrying Embedding Gate

## Target Claims

No claim-ledger target. This is a boundary-adapter open-problem gate only.

## Setup

T504 made the source-category and functoriality burden executable. The follow-up
faithfulness build found that the physical/mirror face distinction is
order-reflected, but the flat D1 capability profile still admits a spurious
pure-physical collapse:

```text
F(W+) -> F(W+0)
```

because two identical physical sites can map many-to-one into one physical
site. T505 then added the observer/system gauge guard and kept current adapter
agreement review-only.

T506 tests a synthetic TaF-side fix: a capability encoding carries exact
dimension atoms and permits only injective, face-preserving morphisms.

## Success Criteria

- The flat D1 baseline is rejected because it admits `F(W+) -> F(W+0)`.
- A dimension-carrying encoding preserves all expected finite source
  inclusions.
- The same encoding blocks non-source-order morphisms, especially
  `F(W+) -> F(W+0)`.
- The physical/mirror face distinction remains protected.
- Cardinality-only, noninjective, post-hoc-retuned, physics-wrong, claim,
  external-publication, and cross-repo shortcuts are rejected or blocked.

## Failure Criteria

- The dimension-carrying encoding still admits `F(W+) -> F(W+0)`.
- Expected source inclusions such as `W+0 -> W+`, `W+ -> W`, or `W- -> W`
  fail.
- The encoding repairs dimension only by losing the mirror/physical face guard.
- The test is used to assert real GU source-category truth, real TI source
  truth, adapter identity, public posture, or cross-repo truth.

## Known Physics Constraints

This gate uses only local synthetic TaF fixtures and the existing mini-T12'
profile convention from the boundary-adapter build. It does not inspect sibling
repos, rerun GU or TI source fixtures, or assert physics content beyond the
review-only local encoding test.

## Contribution Needed

If this gate stays useful, a later packet must supply a real source-owned
dimension semantics, prove an equivalent no-many-to-one condition, and keep
the result review-only until source-category truth is supplied by the owning
surface.
