# T532: Finality-Native Ordering-Fraction Law Starter Router

- [S1: Spacetime As Consensus Envelope](../claims/S1-spacetime-consensus-envelope.md)
- [T528: S1 Finality-Native Generator Preflight](T528-s1-finality-native-generator-preflight.md)

- [T531: Ordering-Fraction Measure-Law Contract](T531-ordering-fraction-measure-law-contract.md)

## Central Question

Can the T531 contract be instantiated as a deterministic starter router for
candidate finality-native ordering-fraction laws, while distinguishing
independently natural candidates from target-import, screen-conditioned, and
post-hoc band-fit candidates?

## Setup

T532 is the first TAF1 starter artifact. It does not try to solve S1 or produce
a new manifoldlike generator. It evaluates a small predeclared candidate menu:

- receipt product-order laws motivated by local finality receipts;
- a Lorentzian `u/v` reference import;
- a T528/T530 screen-conditioned repair;
- a post-hoc repaired-band fit.

The router separates three questions:

1. Does the candidate satisfy the T531 source-contract guardrails?
2. Does it have independent naturality rather than importing the target or
   fitting the repaired screen?
3. Does it actually clear as a real candidate, meaning the current deterministic
   evidence is enough to treat it as more than review-only or negative?

## Success Criteria

- The T531 ordering-fraction route is preserved.
- Target-import, screen-conditioned, and post-hoc band-fit candidates are
  rejected with distinct classifications.
- Independent-naturality candidates are not conflated with target imports or
  post-hoc fits.
- Existing negative evidence from T528/T530 is preserved.
- If no real candidate clears, the result says so explicitly.
- The writeup labels claims as COMPUTED or ARGUED with confidence.
- No S1, claim, canon, public-posture, external-publication, or cross-repo
  movement is made.

## Failure Criteria

T532 fails if:

- it treats the starter router as an S1 result;
- it admits a Lorentzian-reference import as finality-native;
- it admits a screen-conditioned or post-hoc band-fit law;
- it hides T528's negative evidence;
- it reports an independently natural candidate as a real clear without
  sufficient deterministic evidence;
- it moves S1, claim status, Canon Index tiers, canon verdicts, public posture,
  external-publication state, or cross-repo truth.

## Run Command

```bash
python -m models.run_t532
python -m unittest tests.test_t532_finality_native_ordering_fraction_law_starter -v
```

## Boundary

T532 is a deterministic starter router only. It does not derive spacetime,
prove manifoldlikeness, establish a finality-native sprinkling law, repair
T528, reverse T223, promote S1, change claim status, change canon, change
public posture, authorize external publication, or support cross-repo truth.
