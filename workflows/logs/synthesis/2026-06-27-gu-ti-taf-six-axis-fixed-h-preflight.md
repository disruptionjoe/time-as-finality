# GU/TI/TaF Six-Axis Fixed-H Preflight

## Scope

This is a fillable preflight packet for the fixed-H versus H-growing fixture
requested by the GU/TI/TaF reciprocal bridge contract. It does not supply a
physical-source proof, promote Temporal Issuance, promote H7, merge claim
ledgers, or make the cross-repo bridge canonical.

The only admissible purpose of the next artifact is to decide whether a
TI-C020-style candidate has been typed tightly enough to run a fixed-H null
against an H-growing success condition.

- `open-problems/gu-ti-taf-reciprocal-bridge-contract.md`.

## Current Contract Posture

The bridge contract gives the safe cross-repo state:

```text
GU supplies sharper formal gates and absorbers.
TI supplies a formal source candidate under MLTT.
TaF supplies observer-finality, access, colimit, and source/readout discipline.
None of the three currently proves the others.
```

This preflight preserves that posture. A successful packet can authorize a
bounded fixture, not a bridge claim.

## Fillable Six-Axis Packet

The next run must fill this table before any null or success verdict is read:

| Axis | Required fields | Fail-fast condition |
| --- | --- | --- |
| L1 substrate | Hilbert space or source-construction substrate, state space, allowed transformations, resource ledger. | The substrate is named only by analogy or changed after the verdict. |
| L2 observer | Observer class, access boundary, instrument/channel, cadence, record-bearing condition. | Observer access is hidden in prose or treated as universal. |
| L3 pairing | Source-to-record generation map, observable algebra, readout map, admissibility predicate. | Pairing data changes between fixed-H and H-growing arms. |
| L4 causal order | Causal order, access regions, worldlines, volume/counting data, no-global-order convention. | A global present or total commit order is smuggled in. |
| L5 emergence class | Fixed-H null class, H-growing candidate class, class-exit criterion, absorber granted. | The H-growing class is just a relabeling of fixed-H access maps. |
| L6 coordination loop | Identity, overlap, gluing data, observer-domain descent, record/source filtration, comparison readout. | Colimit/descent and filtration are collapsed into one axis. |

Blank or metaphorical fields do not refute the idea. They return the packet for
typing.

## Fixed-H Null

The fixed-H arm must be granted first. It may use any of these equivalent
native forms, as long as they are declared before execution:

```text
fixed Hilbert space H
fixed observable algebra A
fixed channel or instrument I
fixed H_infty with access maps
fixed source construction with observer projections
```

The null succeeds if it reproduces the proposed transition while preserving:

```text
observable types
record maps
perturbation effects
access predicates
admissibility predicates
readout values
same-neighbor freeze fields
```

Allowed fixed-H verdict:

```text
fixed_H_absorbs_transition
```

This verdict means the candidate is a projection, access, channel, or readout
effect inside a fixed structure. It is not a failure of TaF, TI, or GU; it is
the expected absorber for under-typed bridge claims.

## H-Growing Success Condition

The H-growing arm can be scored only after the fixed-H null is specified. It
must show that no fixed-H, fixed-A, fixed-channel, or fixed-H_infty-with-access
model reproduces the transition under the same freeze.

Minimum H-growing witness fields:

```text
H_stage_index
stage_transition_rule
new_observable_or_type
source_admissibility_change
record_generation_change
observer_access_relation
perturbation_response
fixed_H_exclusion_argument
absorber_attempts
```

Allowed H-growing verdict:

```text
H_growing_fixture_survives_preflight
```

This verdict means only that a later executable fixture is well-posed. It does
not mean physical-arrow evidence, source-arrow evidence, or TI-C020 promotion.

The next artifact is accepted as decision-grade only if all of the following
hold:

- All six GU axes are filled with typed data or explicitly marked returned for
  typing.
- The fixed-H null is declared before the H-growing success condition is
  scored.
- The same-neighbor-data freeze is recorded and unchanged across arms.

## Null Or Demotion Conditions

Demote the bridge candidate to null or translation residue if any condition
holds:

- The six-axis table cannot be filled beyond metaphorical labels.
- A fixed-H, fixed-A, fixed-channel, or fixed-H_infty-with-access model
  reproduces the transition.
- The claimed split changes detector cadence, access, gluing, source
  admissibility, or record-generation rules between arms.
- The H-growing arm only renames a changing observer projection.
- TaF colimits are used as spacetime derivation without causal, metric,
  covariance, and locality gates.
- GU observer projection is treated as TI source issuance.
- The packet requires a global commit order or universal present.
- The double diagram collapses horizontal descent and vertical filtration into
  one construction.

Null result language to preserve:

```text
The six-axis packet did not produce a nonabsorbed H-growing fixture. The
candidate remains a fixed-H/readout/access absorber or an under-typed bridge
proposal, with no claim-ledger effect.
```
