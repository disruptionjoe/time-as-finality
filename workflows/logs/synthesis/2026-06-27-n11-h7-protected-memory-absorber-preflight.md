## Plain-English finding

Long-lived or protected storage is not the same thing as impossible physical
deletion. A memory can be scientifically strong while still being the wrong
object for H7.

## Technical conclusion

N11 classifies current protected-memory routes as absorber-owned by default:

- surface and toric-code protection often relies on recovery, decoding, or
  finite barriers;
- higher-dimensional self-correcting examples are stability or scaling results,
  not frozen deletion-reverse witnesses;
- engineered dissipation belongs to the driven-dissipative absorber class;
- symmetry-protected routes need finite enforcement and sector-control audits.

The key technical separation is:

```text
protected storage time != constructor-impossible physical deletion
```
